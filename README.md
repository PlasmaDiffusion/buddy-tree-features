## Buddy Tree Features

# Project Structure

Backend is in (outer) root folder
Front end is in frontend folder

## Back End Commands

# flask run

Start server locally on http://127.0.0.1:5000/

# flask db init

Initialize database

# flask db migrate

Migrate postres database

# flask db upgrade

Migrate postres database

## Front End Commands

# npm start

Run the React front end on localhost:3000

# npm test

Run front end unit tests
